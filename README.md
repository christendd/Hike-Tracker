# Ramble

A hiking tracker that lets you track your completed hikes with the information you want. Users can add new hikes, edit old hikes or delete hikes.

## Technologies Used

* HTML 5
* JavaScript
* CSS
* Google Fonts
* MongoDB
* Heroku
* Express
* Node.js
* Mongoose
* EJS
